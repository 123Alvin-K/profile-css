HTML as One of My Core Skills

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages.

HTML does not function as a programming language but as a markup language that structures content on the web using elements and tags.

An HTML element is made up of a start tag, content, and an end tag.

Tables organize data in rows and columns.

Common table elements include:

  • <table> – Defines the table.
  • <tr> – Table row.
  • <th> – Table header cell.
  • <td> – Table data cell.

    • Forms

    Forms allow users to input data.

    Important form elements include:

    • <form> – Defines a form.
    • <input> – Allows users to enter information.
    • <label> – Labels form fields.
    • <textarea> – Multi-line input.
    • <button> – Submits the form.
      • Semantic Elements

      Semantic HTML elements clearly describe the meaning of content.

      • <header> – Introductory content.
      • <nav> – Navigation links.
      • <section> – Thematic grouping.
      • <article> – Independent content.
      • <footer> – Footer information.
